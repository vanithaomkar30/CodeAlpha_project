# CodeAlpha_project
# 🐍 Python Mini Projects Collection

This repository contains a collection of beginner-friendly Python projects built to practice core programming concepts such as loops, functions, file handling, regular expressions, and basic algorithms.

## 📁 Projects Included

### 🤖 Simple Rule-Based Chatbot
A basic chatbot that responds to predefined user inputs like "hello", "how are you", and "bye".

**Features:**
- Uses if-elif conditions for response handling
- Implements loops for continuous interaction
- Demonstrates basic input/output handling in Python

### 🧠 Task Automation Script
A simple automation project that performs repetitive tasks efficiently.

**Features:**
- Uses Python modules such as `os` and `shutil`
- Uses the `re` (Regular Expressions) module to find and extract patterns like email addresses from text files
- Demonstrates file handling and automation techniques
- Helps reduce manual effort in repetitive tasks

### 📈 Stock Portfolio Tracker
A basic stock portfolio management application for tracking investments.

**Features:**
- Stores stock information
- Calculates portfolio value
- Uses dictionaries and arithmetic operations
- Demonstrates data management in Python

### 🎮 Hangman Game
A command-line word guessing game where players try to guess a hidden word.

**Features:**
- Uses loops and conditional statements
- Implements string manipulation
- Enhances logical thinking and problem-solving skills

## 🛠️ Technologies Used

- Python 3
- Built-in Modules:
  - os
  - shutil
  - re (Regular Expressions)
